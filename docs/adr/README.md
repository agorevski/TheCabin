# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records (ADRs) for The Cabin project.

## What is an ADR?

An ADR is a document that captures an important architectural decision made along with its context and consequences.

## Template

See [ADR-TEMPLATE.md](ADR-TEMPLATE.md) for the template used to create new ADRs.

## Creating a New ADR

1. Copy `ADR-TEMPLATE.md` to a new file named `XXXX-title-with-dashes.md`
2. Fill in the template sections
3. Update this README with the new ADR entry
4. Submit a pull request for review
